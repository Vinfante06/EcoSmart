# EcoSmart Budget

La documentación principal del proyecto está en el archivo `../README.md`.

Esta carpeta contiene la aplicación Django activa:

- `EcoSmart/`: configuración del proyecto.
- `budget/`: modelos, formularios, servicios, reportes, vistas, rutas y pruebas.
- `templates/`: plantillas HTML usadas por la aplicación.
- `static/`: estilos e imágenes.

Para ejecutar el proyecto desde la raíz del repositorio:

```bash
./.venv/bin/python manage.py runserver
```
